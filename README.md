# Сайт словаря

Локальный сайт для повторения английских слов из файла `Новые слова.md`.

Как открыть:

1. Открой файл `index.html` в браузере.
2. В режиме `Карточки` нажимай на карточку, чтобы увидеть перевод.
3. Кнопка `Отметить выученным` сохраняет прогресс в браузере.
4. Кнопка `+ Добавить слово` открывает форму для новых слов. Добавленные слова тоже сохраняются в браузере.
5. Кнопка `Редактировать карточку` в режиме карточек открывает эту же форму для правки текущего слова.

Если обновишь `Новые слова.md`, сайт нужно пересобрать, чтобы новые слова попали в `app.js`.
